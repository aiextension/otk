Quick Start

Installation
# Clone the repository
git clone https://github.com/aiextension/open-otk.git
cd open-otk

# Install Open OTK
pip install -e ".[scraper]"

# Launch from anywhere
otk
Basic Usage
from otk import OllamaClient

client = OllamaClient()
response = client.generate("llama2", "Tell me a joke")
print(response)
